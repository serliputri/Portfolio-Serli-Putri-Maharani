Portfolio

Project Akhir Semester 1: Kasir

Aplikasi kasir ini tentunya sangat sederhana karena menggunakan bahasa pemrograman java yang sangat dasar. Aplikasi kasir ini merupakan POV dari kasir. Kasir dapat melihat barang yang telah diinputkan, melakukan transaksi, melakukan re-stock barang yang baru datang, dan melihat riwayat transaksi.

Project Akhir Semester 2: Online Shop

Aplikasi project ini berbasis basis data dan merupakan bentuk implementasi dari platform online shop oren. Aplikasi ini menggunakan aplikasi MySQL untuk menjalankan source code. Dimana pelanggan dapat melakukan daftar akun baru, login akun, mencari barang melalui inputan, melihat detail barang dan detail profil, dan melihat riwayat transaksi.

Project Akhir Semester 3: Tiket Kereta Api

Aplikasi kasir penjualan tiket kereta api ini menggunakan bahasa pemrograman Java berbasis GUI dan database. Aplikasi ini dapat melihat dan mengedit daftar kereta, menginputkan nama penumpang, mencari data kereta atau data penumpang, dan juga dapat melakukan cetak tiket.

Project Akhir Semester 3: Rental Mobil

Aplikasi ini adalah aplikasi rental mobil berbasis website sederhana. Aplikasi ini dapat melakuakan daftar dan login user, lalu user memilih mobil sesuai kebutuhan, lokasi peminjaman, dan jangka waktu peminjaman, lalu user melakukan pembayaran dapat melalui e-wallet, transfer, dan tunai.

Project Akhir Semester 4: Katering

Aplikasi ini adalah aplikasi katering berbasis website. Aplikasi ini hampir mirip dengan platform Gfood/Sfood, yaitu kita menyediakan berbagai macam makanan yang diproduksi sendiri. User dapat melihat menu, memesan makanan, melakukan pembayaran, dan melihat history pesanan.

Project Akhir Semester 5: Peminjaman Buku berbasis OCR

Aplikasi ini adalah aplikasi peminjaman buku berbasis mobile dan OCR. OCR dalam aplikasi ini adalah untuk mendeteksi identitas user yang akan di scan untuk user peminjaman buku, sehingga kita tidak perlu input data-data manual yang terkadang menyebabkan kesalahan dalam penulisan.
